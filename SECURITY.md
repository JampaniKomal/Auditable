# Security Policy

Auditable is a deliberately insecure training repository. The vulnerabilities in the scenarios are intentional and part of the exercise.

## Supported Scope

- Documentation issues
- Repository hygiene and packaging issues
- Broken scenario builds that prevent local training
- Problems with the intended audit workflow

## Out of Scope

- Requests to remove the intentional weaknesses from the training scenarios, unless the request is specifically about repository quality or documentation

## Reporting a Problem

If you find an unintended issue in the repository, open an issue or contact the maintainers with:

- Scenario name
- Exact file or command
- Reproduction steps
- Expected versus actual result

For urgent repository safety concerns, prefer a private disclosure path rather than a public issue.
